🧪 TEST A2 – Indexation par Bing uniquement

🔍 Objectif du test

Ce test vise à vérifier si SearchGPT est capable d’accéder à une page web visible uniquement par Bing. Le fichier robots.txt interdit l’accès à tous les bots sauf Bingbot. Le contenu contient une phrase absurde unique qui ne figure nulle part ailleurs sur le web.

🧾 Phrase absurde à détecter

Le sémaphore en mousse de Nogent-le-Rotrou déclame des poèmes tibétains à l’envers pendant les marées vertes quantiques.

📁 Configuration robots.txt

User-agent: Bingbot
Allow: /

User-agent: Googlebot
Disallow: /

User-agent: GPTBot
Disallow: /

User-agent: *
Disallow: /

📌 Résultat attendu

La phrase ci-dessus ne devrait être visible que si SearchGPT utilise Bing comme moteur de recherche.
